Applicaton Files

OnSag uses a few different file types to keep everything straight.

Config File

The config file holds the user settings, such as:

  • Window size and layout
  • Unit System
  • Plot colors

The file location depends on the platform and user.

  • Linux: $HOME/.config/otls/on-sag/onsag.conf
  • Windows: %AppData%\OTLS\OnSag\onsag.conf


If this file is not available on startup, a new one will be created.

Document Files

OnSag uses document files so projects can be closed and reopened at a later time. Document files have an extension of '.onsag'. These files can be stored anywhere on the system.

The document file stores details for the sag spans, including:

  • Cable properties
  • Structure attachment locations
  • Sagging method information
  • Notes
