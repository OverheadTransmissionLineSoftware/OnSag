Editing Sag Spans

Sag span data is stored in a document (.onsag) file. Editing starts in the Edit Pane. Sag spans can be added, removed, copied, modified, or moved within the list.

To trigger an editing action, either:

  • Select a sag span and click a button in the edit pane.
  • Right click on a sag span and select a menu entry.

Sag Span Editor

The sag span editor is launched when an individual sag span needs to be edited. There are multiple tabs for the data.

Cable

This tab shows all cable related information.

Five (5) entries are provided for entering cable tension. All of the entries need to be filled. For cables that are sensitive to temperature, the horizontal tensions should decrease. For cables that maintain a constant tension for all temperatures, enter multiple temperatures but keep the same tension. Additional information is provided in the Calculations->Catenary help entry.

The corrections allow the cable tension to be modified from the above entered values. Additional information is provided in the Calculations->Catenary help entry.


Structures

This tab shows all structure related information.

The structure and attachment are for documentation purposes, and don't affect the analysis.

A single attachment position is modeled for the structure. The X entry is the horizontal position and Y entry is vertical position.

Sagging

This tab shows all sagging method related information.

A method is selected using the leftmost control. Once selected, options for the specific sag method are unlocked for editing.

  • Transit - Additional information is provided in the Calculations->Transit Sagging help entry.
  • Dynamometer - Additional information is provided in the Calculations->Dynamometer Sagging help entry.
  • Stopwatch - Additional information is provided in the Calculations->Stopwatch Sagging help entry.

The Temperature section is where the sagging temperatures are set. The base temperature is the expected/average temperature for the day, while the interval temperature sets the temperature range. A total of five (5) sagging temperatures are calculated:

  • Sagging Temp 1 = Base Temp - (2 x Interval Temp)
  • Sagging Temp 2 = Base Temp - (1 x Interval Temp)
  • Sagging Temp 3 = Base Temp
  • Sagging Temp 4 = Base Temp + (1 x Interval Temp)
  • Sagging Temp 5 = Base Temp + (2 x Interval Temp)

Notes

This tab is used for adding notes for the sag span. Names of the sagger/checker, reference drawings, and assumptions could be useful to track.
