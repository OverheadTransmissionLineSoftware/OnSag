Reports

The analysis data can be viewed in the Report Window.



Report Table

The report table displays data based on the sagging method.

Transit

The transit sagging method will display the following:

  • Temperature - cable temperature (°C or °F)
  • H = catenary horizontal tension (N or lbs)
  • S = catenary sag (m or ft)
  • CF = control factor
  • A = transit angle (°)
  • t = target dimension (m or ft)

Dynamometer

The dynamometer sagging method will display the following:

  • Temperature - cable temperature (°C or °F)
  • H = catenary horizontal tension (N or lbs)
  • S = catenary sag (m or ft)
  • T = dynamometer tension (N or lbs)

Stopwatch

The stopwatch sagging method will display the following:

  • Temperature - cable temperature (°C or °F)
  • H = catenary horizontal tension (N or lbs)
  • S = catenary sag (m or ft)
  • WS = traveling wave speed (m/s or ft/s)
  • L = catenary length (m or ft)
  • Time = return wave time (s)

Sorting

The results shown in the table can be sorted. Clicking on a column will trigger a sort, and clicking again will alternate ascending/descending.

To clear sorting, right click on any column and select 'Clear Sort'.

Exporting

Currently, the only way to export the results is copy/paste. Right click on the table columns or rows and select the copy option that fits best.
