Windows

Here is a snapshot of a default window layout. The window positions can be resized but not rearranged.




Edit Window

This window allows the sag spans to be edited or activated for an analysis. There can be many sag spans in this window, but only one can be selected at a time.

Results Window

This window displays the results from the sagging analysis.

Plot Window

This window contains a plot that shows the cable position and any sagging method details. The information plotted will match the selection of the Results Window.

Log Window


This window displays errors or warnings that are encountered. If something isn't working right, this should be the first place to check. Select View->Log from the menu to toggle window visibility.
