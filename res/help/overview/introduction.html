Introduction

OnSag is a program developed by Overhead Transmission Line Software (OTLS). It was built to provide installation crews with a tool to perform sagging calculations.

The sagging methods include:

  • Transit
  • Dynamometer
  • Stopwatch

Some of the user interface features include:

  • Load/save files with multiple span configurations
  • Undo/redo actions
  • Plotting
  • Printing
